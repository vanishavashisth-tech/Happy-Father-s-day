# Happy Father's Day Website ❤️

A beautiful, responsive, and interactive Father's Day landing page designed to celebrate fathers and father figures. The website features heartwarming images, smooth animations, an interactive greeting card, inspirational quotes, and a photo gallery.

## ✨ Features

- 🎉 Responsive design for mobile, tablet, and desktop
- ❤️ Elegant Father's Day themed UI
- 🖼️ Hero section with father-child imagery
- 📸 Interactive image gallery with hover effects & lightbox
- 💌 Personalized greeting card section (live preview)
- 🌟 Inspirational Father's Day quotes
- 🎨 Modern design with gradients and glassmorphism
- ⚡ Smooth scrolling and scroll-triggered animations
- 💛 Floating heart particles background
- ⏳ Live countdown to next Father's Day
- 📱 Mobile hamburger navigation

## 🛠️ Technologies Used

- HTML5
- CSS3 (Glassmorphism, CSS Animations, Grid & Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Dancing Script, Inter)

## 🚀 Getting Started

### Run Locally
```bash
git clone https://github.com/your-username/happy-fathers-day.git
cd happy-fathers-day
# Open index.html in your browser
```

### Deploy to Vercel
```bash
npm i -g vercel
vercel
```
Or drag the folder into [vercel.com/new](https://vercel.com/new) — no config needed.
